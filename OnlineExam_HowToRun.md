## How to Run

1. **Compile the code:**
   ```bash
   javac OnlineExam.java
   ```

2. **Run the application:**
   ```bash
   java OnlineExam
   ```

3. **Take the exam:**
   - Log in and answer the questions.
   - Submit before the timer runs out.
